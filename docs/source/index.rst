Welcome to Kai's Machine Learning Laboratory!
===================================

Hello there! In this repository, you'll find useful information on machine learning and artificial intelligence. 

Want to know what we can do with machine learning? Head over to the *Use Cases* page. If you're a developer, head over to the *Resources* section for useful information for your development works. Head over to the *Projects* page to find out more about projects utilising machine learning in NIE.
